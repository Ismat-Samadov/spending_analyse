**Advanced Prompt: Presentation‑Style README**

Role: You are writing a presentation narrative for a personal savings strategy based on spending data.

Task:
- Write a polished README that reads like a business presentation, not a report.
- Use first‑person perspective (owner voice).
- Include intro, key findings, sections tied to charts, and a concise strategy summary.

Style Constraints:
- No technical jargon or implementation details.
- No “you should.”
- Clear, executive tone with short sections.

Structure:
1. Title + Intro
2. Key Findings (bulleted)
3. 4–6 sections with:
   - Chart(s)
   - What it shows
   - Why it matters
   - How it guides decisions
4. Saving Strategy Summary
